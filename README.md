Foram utilizadas as tags: < h1 >, < h2 >, < h3 >, < h4 >, < p >, < b >, < i >, < u >, < strong >, < em >, < a >, < img >, < br >, < ul >, < il >, < link >, < header >, < main >,< footer >, < section >
